# Excel to Json

# Pre-requisites
- Install [Node.js](https://nodejs.org/en/)


# Getting started
- Clone the repository
```
git clone https://github.com/pgen0x/excel-to-json.git
```
- Install dependencies
```
cd excel-to-json
npm install
```
- Build and run the project
```
node app.js
```
  Navigate to `http://localhost:3000`
